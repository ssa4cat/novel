小說後台工具

這是一個靜態管理頁。你可以選作品、填章節資訊、直接貼內容或上傳 txt，系統會幫你產生可貼進小說頁的章節物件。它不會自動寫回 GitHub，但能大幅降低你手動整理格式的成本。

章節資料輸入

如果你的文字檔都是 Big5，就保持右邊編碼選單為 Big5（預設）。若遇到亂碼，再改成 UTF-8 重新讀取。
使用方式：先填好欄位，按「產生章節物件」，再把右側輸出的內容貼進對應小說頁的 articlesData 陣列中。

輸出結果

建議維護方式：每新增一章，就把這裡輸出的物件貼到對應小說頁的 articlesData = [ ... ] 裡面。
未來如果你要，我可以再幫你進一步做成「讀取 JSON / TXT 檔」的版本，讓內容不必再塞進 HTML 裡。
