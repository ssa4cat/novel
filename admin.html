小說後台工具

這是一個靜態管理頁。你可以選作品、填章節資訊、直接貼內容或上傳 txt。系統會幫你產生章節 HTML 檔與 JSON 索引項目，並提供下載，方便你放進 chapters/ 與 data/ 架構中。

章節資料輸入

如果你的文字檔都是 Big5，就保持右邊編碼選單為 Big5（預設）。若遇到亂碼，再改成 UTF-8 重新讀取。
這個後台會幫你自動把純文字轉成段落 HTML，並產出 JSON 索引項目。你只要把下載好的檔案放到對應資料夾，再把 JSON 項目貼進對應作品的索引檔即可。
